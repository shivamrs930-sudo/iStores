# iStore – Apple Inspired E-Commerce Website

iStore is a Java web application inspired by Apple's online store.  
Users can browse products, add items to cart, and place orders.

## Technologies Used
- Java
- JSP
- Servlets
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat

## Features
- Product catalog (iPhone, MacBook, iPad)
- Add to cart functionality
- Checkout system
- Order details stored in MySQL database
- Clean Apple-style UI

## Project Structure
src/
  dao/
  model/
  servlet/

webapp/
  index.jsp
  product.jsp
  checkout.jsp
  success.jsp
  style.css

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat server
3. Setup MySQL database
4. Run project on Tomcat
